# ZhiHuDailyNews
获取知乎日报消息的app，主页显示今天及最近6天的日报消息，下拉可刷新；右上角图标点击显示日历，以日历的形式获取过往的日报消息，点击即可获取所选日期的消息，最后点击具体的消息显示内容。附上apk下载链接，有兴趣的话可以试试看哦~（知乎日报+知乎，效果更佳啊）—— http://pan.baidu.com/s/1dEReaR3
